# Aees bot discord
Bot qui gère des synthèses via discord

## Installation

- Installer python 3.10 minimum.
- Installer les requirements python.
- Remplir les informations dans le fichier TOKEN.
- Démarrer le bot

```bash
pip install -r requirements.txt
```
